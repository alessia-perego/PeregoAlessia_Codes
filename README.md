 :telescope: **Summary**
- 

In this repository, there are two folders with different projects: 
- :bookmark_tabs: in [Astrostatistics_MachineLearning](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning): there are some codes from the astrostatistics and machine learning course. There are different models for the analysis.
  For example:
    - [L10.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L10.ipynb) Markov chain;
    - [L11.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L11.ipynb) emcee;
    - [L12.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L12.ipynb) Dynesty;
    - [L13.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L13.ipynb) dimensionality reduction;
    - [L14.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L14.ipynb) clustering;
    - [L17.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L17.ipynb) and [L18.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L18.ipynb) regression;
    - [L19.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L19.ipynb) unsupervised classification;
    - [L20.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L20.ipynb) ROC-AUC and Decision Tree;
    - [L22.ipynb](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/Astrostatistics_MachineLearning/L22.ipynb) Neural Network.

- :bookmark_tabs: in [NBody_2GalaxyMerger](https://github.com/alessia-perego/PeregoAlessia_Codes/tree/main/NBody_2GalaxyMerger):This is a project done for the course of "Dynamics of stellar systems" in which I initialized two galaxies, described as plummer distribution. The process simulated is cannibalism: one big stable system in which a small one is merging. The evolution was done with a TreeCode. The folder's files are the codes for the [initial conditions] (https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/NBody_2GalaxyMerger/IC_two_galaxies_plot.py) and two files for the plots ([1](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/NBody_2GalaxyMerger/Plot_tot_analysis.py) and [2](https://github.com/alessia-perego/PeregoAlessia_Codes/blob/main/NBody_2GalaxyMerger/Plot_two_galaxies_CM.py)).
